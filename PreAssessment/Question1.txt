Flip Key – Logical Problem Solving 

Problem Overview
A computer science instructor is designing a tool to help students understand ASCII values and string manipulation.

This utility removes characters with even ASCII codes from a word and reverses the remaining characters, helping students explore how data can be transformed for security and encoding purposes.

Method
public string CleanseAndInvert(string input)
This method takes a word as input and performs a series of transformations to return a customized string.

Input Validation Rules
The input must not be null and must contain at least 6 characters.

If not, return an Invalid Input.
The input must not contain spaces, digits, or special characters.

If it does, return an Invalid Input.
Password Generation Logic
If the input is valid:

Convert the input to lowercase.
Remove all characters whose ASCII values are even numbers.
Reverse the remaining characters.
In the reversed string, convert characters at even positions (0-based index) to uppercase.
Return the generated key.

public class Solution{
     public String CleanseAndInvert(String input) {
          if(input == null || input.length()<6){
               return "Invalid Input";
          }

          for(char ch : input.toCharArray()){
               if(!Character.isLetter(ch)){
                    return "Invalid Input";
               }
          }

          input = input.toLowerCase();

          StringBuilder filtered = new StringBuilder();
          for(char ch : input.toCharArray()){
          if(ch % 2 != 0){
         filtered.append(ch);
          }
                    }
                    filtered.reverse();
                    for(int i = 0 ; i < filtered.length(); i++){
                         if(i % 2 == 0){
                              filtered.setCharAt(i , Character.toUpperCase(filtered.charAt(i)));
                         }
                    }
                    return ""+filtered;
     }
}
